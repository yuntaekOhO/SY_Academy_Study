(수석입학)
이력 웹툰작가

이말년씨리즈 : 데뷔작 야후에서 연재 시작하고 네이버로 이적
(2009년1월 ~ 2012년2월)

이말년 서유기 : 코믹물임에도 서유기원전에 가까울 정도로 고증이 잘되어있는게 특징
(2013년12월 ~ 2016년9월)

이 외에도 "이말년 수필", "이말년씨리즈2018" 등의 작품으로 2015년 ~ 2018년까지 웹툰을 연재하다가
이말년씨리즈2018을 마지막으로 사실상 웹툰 작가 생활을 은퇴한다.

유튜버,방송인

2011년 KBS "유희열의 라디오 천국"의 게스트로 첫 출연
2016년 MBC "무한도전" 웹툰 특집 게스트 출연
2018년 SBS "배성재의 텐" 고정 게스트로 출연 후 2022년 현재까지 출연중
2020년 KBS "그놈이 그놈이다" 특별출연
2022년 MBC "놀면 뭐하니?" 게스트, 일일MC로 출연

다음팟에서 스트리머를 처음 시작했다. 이때부터 침착맨이라는 이름 사용했다.
개인 방송을 시작하게 된 계기는 어렸을때부터 게임을 좋아했는데 혼자서만 게임을 하다 지겨워지니,
여럿이서 얘기하면서 해보면 어떨가 싶어서 시작한것이 시초였다고한다.
시청자가 늘면서 아프리카TV로 이적하고 2015년에 다시 트위치TV로 이적
현재까지 트위치TV에서 개인방송을 진행하고 있고 현재 개인 유튜브 채널도 운영하고 있으며
2022년 5월 27일 기준으로 구독자 수는 162만명이다.
